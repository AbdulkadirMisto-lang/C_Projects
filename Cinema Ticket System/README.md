# 🎬 Cinema Ticket System in C

A simple file-based Cinema Ticket Booking System written in C.  
It allows users to book tickets, add extras (popcorn, juice), choose payment methods, confirm payments, display bookings, and delete tickets.

---

## ✨ Features
- 🎟️ Book tickets for movies  
- 🍿 Add extras like popcorn and juice  
- 💰 Automatic price calculation  
- 💳 Choose payment method (Cash, Credit Card, Mobile Payment)  
- ✅ Confirm payment before completing booking  
- 📋 Display all bookings with payment status  
- ❌ Delete tickets by ID  
- 💾 Persistent storage using `tickets.dat`
